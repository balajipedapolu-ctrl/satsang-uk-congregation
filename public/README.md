# public/ — static files

Anything in this folder is served at the site root.

## Header banner image

The homepage header shows a banner of the revered figures
(Sree Sree Borda, Sree Sree Thakur, Sree Sree Baroma, Sree Sree Acharyadev).

**To show it, place the image here named exactly:**

```
public/founders.jpg
```

Until that file exists, the banner hides itself automatically (the rest of the
page is unaffected). To change the image later, just replace this file with the
same name.

Recommended width: ~2000px, landscape (roughly 2.8 : 1).
